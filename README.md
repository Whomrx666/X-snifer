# X-snifer
![X-snifer preview](X-snifer.jpg)

## introduction
X-snifer is a versatile tool designed for scanning and gathering information from a website and simplifying various website analysis and network reconnaissance tasks.

## Instalations
```
$ pkg update -y && pkg upgrade -y
$ apt update -y && apt upgrade -y
$ pkg install git
$ pkg install python -y
$ git clone https://github.com/Whomrx666/X-snifer.git
$ cd X-snifer
$ pip install -r requirements.txt
$ python3 X-snifer.py
```

## Instructions
- **first**: Install tools according to the instructions above.
- **second**: Select one of the tools you want to use according to the numbers.
- **third**: Follow the directions from the tool you have chosen and enter according to the directions.
- **last**: The tool will automatically perform a scanner.

## Feature
- **WP Plugin Scanner:** Identify WordPress plugins and their versions.
- **Path Enumeration:** Discover hidden paths and directories on a website.
- **WP Enumeration:** Enumerate WordPress users, themes, and more.
- **IP Scanner:** Perform IP address reconnaissance and mapping.
- **Ports Scanner:** Scan for open ports on a target IP address.
- **Subdomain Scanner:** Find all subdomains of a website.
- **Wayback URLs:** URL archive collector.
